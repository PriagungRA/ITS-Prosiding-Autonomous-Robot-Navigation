# ITS Conference Proceeding

## Autonomous Wheeled Robot Navigation Based on Fuzzy Logic Using LiDAR and Thermal Camera

This repository contains the LaTeX source code of the conference proceeding manuscript derived from the undergraduate thesis research on autonomous robot navigation.

### Author

Priagung Ramadhan Alfalakhi
Department of Electrical Engineering
Institut Teknologi Sepuluh Nopember (ITS)

### Research Topics

* Autonomous Robot Navigation
* LiDAR Perception
* Thermal Camera Perception
* Sensor Fusion
* Mamdani Fuzzy Logic
* Artificial Potential Field (APF)
* ROS Navigation Stack
